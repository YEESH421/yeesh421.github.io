A simple one-page web app built on react. It features a game of tic-tac-toe, with an option to play against a cpu player that uses a minimax tree to make moves.

GroupMe Bot

I made a GroupMe bot in python. It is hosted on Heroku, a cloud application platform. The bot sends inspirational quotes at a specified time based on a clock, but the problem is that Heroku apps fall asleep after inactivity. I would have to input my credit card info to gain access to the Heroku clock system which would wake up my app. It also sends inspirational quotes when prompted, and does Wikipedia searches returning the first two sentences on Wikipedia on the serached topic.
